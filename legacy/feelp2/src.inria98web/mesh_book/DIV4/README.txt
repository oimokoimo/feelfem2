有限要素法メッシュ４分割ツール div4 の利用方法               '95/5/24
                                                 NIS 渡部弘(8-339-687)

１．インストール方法

(1) ディレクトリ  DIV4/ をディレクトリごと自分の作業ディレクトリにコピーする。
(2) make をする。


２．使い方

(1)FEELのメッシュジェネレータを用いて、オリジナルのメッシュファイル MESH.PLN
   を作成する。このMESH.PLNは３角形メッシュ、４角形メッシュのどちらでもよいが、
   混在していてはいけない。

(2)div4と入力する。するとMESH.PLNの４分割メッシュが生成され、MESH.PLNが書き
   変えられる。従って、オリジナルのメッシュファイルが必要なら、あらかじめ
   別の名前でコピーしておく必要がある。


３．注意事項

(1)本ルーチンはかなりのメモリを必要とします。メモリは３０MB程度必要です。
   このメモリが確保できない場合はコアダンプします。もしメモリを減らしたければ、
   ファイル main.f 冒頭の

      PARAMETER( KTJ = 200000 )
      PARAMETER( KTE = 200000 )

   及びファイル renum4.f 冒頭の

      parameter ( nomax = 200000 )
      parameter ( kcm = 15 )

   の20000という数字を小さくして、リコンパイルします。ちなみに、KTJとnomaxは
   最大節点数、KTEは最大要素数です。

(2)本ルーチンの実行時間は1000節点の３角形要素で５５秒、2000節点で２３０秒です
   (ただし、EWS4800/330)。時間は節点数の２乗に比例します。
